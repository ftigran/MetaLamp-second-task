# fsd-second-task
Второе задание в рамках программы обучения компании Fullstack Development.
#Развёртывание
`npm i`
`npm run start` - запуск dev server
`npm run dev` - сборка в режиме разработки
`npm run build` - сборка в режиме продакшн
